Welcome to xs’s documentation!¶

xs is a dynamically typed, dynamically scoped, concatenative array language inspired by kdb+/q and released into the public domain. The interpreter and builtin functions are written in OCaml. Check out the project page on github if you wish to browse the source, open an issue or pull request.
